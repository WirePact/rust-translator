Expand description

Translator for ingress and egress communication. This trait is used in the respective gRPC servers to translate authentication data into a signed JWT and vice versa.

Inbound communication translator. Used to transform a signed JWT (if any) into the corresponding authentication data. If no JWT is available or no subject_id can be parsed, the method will not be called and the IngressResult will be IngressResult::forbidden.
